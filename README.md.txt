# Binance Futures Order Bot (CLI)

A command-line based trading bot built using Python for Binance USDT-M Futures (Testnet).
The bot supports market orders, limit orders, logging, and an advanced OCO (Take-Profit + Stop-Loss) strategy.

---

## Features

- Market Orders (BUY / SELL)
- Limit Orders
- Advanced OCO Orders (Take Profit + Stop Loss)
- Input validation via CLI arguments
- Structured logging (`bot.log`)
- Binance Futures Testnet support (no real money)
